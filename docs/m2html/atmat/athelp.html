athelp

PURPOSE

ATHELP generates the list of Accelerator Toolbox functions

SYNOPSIS

function athelp

DESCRIPTION 

ATHELP generates the list of Accelerator Toolbox functions

  EXAMPLES

  2. for selected help, use help directory where directory is

class Wake(srange)#

Bases: object

Class to generate a wake object The wake object is define by its srange, specified at initialization, and DX, DY, QY, Z corresponding to transverse dipoles and quadrupoles and longitudinal

The srange is common to all components and cannot be changed once initialized, all added component are resampled to the srange

usage: wake = Wake(srange) wake.add(WakeType,WakeComponent, *args, *kwargs)

Component are WakeComponent.FILE (import from file), WakeComponent.TABLE (provide vectors), WakeComponent.RESONATOR (analytical resonator), WakeComponent.RESWALL (transverse RW)

Components are retrieved with Wake.DX for example

static build_srange(start, bunch_ext, short_step, long_step, bunch_interval, totallength)#

Function to build the wake table s column. This is not the slicing but the look-up table, however it generates data where bunches are located to avoid using too much memory to store the table.

PARAMETERS
start starting s-coordinate of the table

(can be negative for wake potential)

bunch_ext maximum bunch extension, function

generates data at +/- bunch_ext around the bucket center

short_step step size for the short range wake table long_step step size for the long range wake table bunch_interval minimum bunch interval data will be generate

for each bunch_inteval step

totallength total length of the wake table, has to contain

the full bunch extension

OUTPUT

srange vector of s position where to sample the wake
